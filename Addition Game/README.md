

# 🔢 Addition Game – JavaScript DOM Project

This project is part of my **JavaScript learning journey**, where I built a simple interactive math game using DOM manipulation and event handling.

---

## 🚀 Project Overview

The **Sum Game** generates two random numbers, and the user has to enter the correct sum.

* Click **Check** → verifies the answer
* Click **Restart** → generates new numbers

---

## 🎯 Features

* 🔢 Random number generation using `Math.random()`
* 🧠 User input validation
* ✅ Instant feedback (Correct / Try Again)
* 🎨 Dynamic UI updates (background color changes)
* 🔄 Restart functionality to generate new questions

---

## 🧠 Concepts Practiced

* DOM Selection (`getElementById`)
* Event Handling (`addEventListener`)
* Type Conversion (`Number()`)
* Dynamic content update (`textContent`)
* Conditional logic (`if-else`)
* Functions for reusable logic

---

## 💻 How It Works

1. Two random numbers are generated
2. User enters the sum in the input box
3. On clicking **Check**:

   * If correct → success message shown
   * If wrong → retry message shown
4. On clicking **Restart**:

   * New random numbers are generated
   * Input and result are reset.
