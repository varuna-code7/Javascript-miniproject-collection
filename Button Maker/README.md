# 🎨 Button Maker – JavaScript DOM Project

This project is part of my **JavaScript learning journey**, where I built a dynamic UI to customize a button’s style using user inputs.

---

## 🚀 Project Overview

The **Button Maker** allows users to:

* Enter style values (color, size, padding, etc.)
* Click the **Apply** button
* Instantly see the styled button update

---

## 🧠 Concepts Practiced

* DOM Selection (`getElementById`)
* Event Handling (`addEventListener`)
* Dynamic styling using `.style`
* Handling user inputs
* Understanding CSS behavior (including `:focus`)
* Overriding Bootstrap styles

---

## ✨ Features

* 🎯 Apply custom styles to a button
* 🎨 Change:

  * Background color
  * Font color
  * Font size
  * Font weight
  * Padding
  * Border radius
* 📢 Displays success message after applying styles
* 🎯 Uses Bootstrap for layout and design

---

## ⚠️ Important Learnings

* Bootstrap applies default styles like `input:focus` (blue highlight)
* CSS from external libraries can override your styles
* DOM elements must exist before accessing them in JavaScript

## 💻 How It Works

1. User enters values in input fields
2. Clicks the **Apply** button
3. JavaScript reads input values
4. Styles are applied dynamically to the button using `.style`
