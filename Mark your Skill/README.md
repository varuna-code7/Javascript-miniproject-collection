

#  Checkbox Skills List

This project is part of my **JavaScript learning journey**, where I practiced creating dynamic UI elements and handling user interactions using DOM manipulation.

---

## 🚀 What This Project Does

* Dynamically creates a list of skills using JavaScript
* Adds a checkbox and label for each skill
* Changes the label color when the checkbox is selected/unselected

---

## 🧠 Concepts Practiced

* DOM Manipulation (`createElement`, `appendChild`)
* Looping through arrays (`for...of`)
* Event Handling (`addEventListener`)
* Checkbox state handling (`checked`)
* Linking label with input using `for` attribute
* Dynamic ID generation

## 💻 How It Works

1. A list of skills is stored in an array
2. JavaScript loops through the array
3. For each item:

   * Creates `<li>`, `<input>`, and `<label>`
   * Assigns unique IDs
   * Links label to checkbox
4. Adds an event listener to detect checkbox changes
5. Updates label color based on checkbox state
