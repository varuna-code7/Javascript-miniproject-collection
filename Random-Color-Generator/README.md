## 📌 Random Color Generator

A simple and interactive web application that generates a random background color when the user clicks a button.

---

## 🚀 Features

* 🎨 Generates random colors from a predefined array
* 🖱️ Button click changes the background instantly
* 📱 Responsive layout using Bootstrap
* ⚡ Lightweight and fast

---

## 🛠️ Tech Stack

* **HTML**
* **CSS**
* **JavaScript**
* **Bootstrap 4**

---

## 📸 How It Works

1. User clicks the **"Click Me"** button
2. A random index is generated
3. A color is selected from the array
4. Background color changes dynamically
