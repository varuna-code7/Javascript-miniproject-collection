## Tabs UI using JavaScript (DOM Manipulation)

This project is part of my **JavaScript learning journey**, where I built a simple Tabs UI to understand how to dynamically control content visibility using DOM manipulation and event handling.

---

## 🚀 What This Project Does

* Displays information about a place (Varanasi) using tabs
* Allows users to switch between:

  * About
  * Time to Visit
  * Attractions
* Shows only one tab at a time
* Highlights the selected tab button

---

## 🧠 Concepts Practiced

* DOM Selection (`getElementById`)
* Event Handling (`addEventListener`)
* Dynamic UI updates using JavaScript
* CSS manipulation using `style` property
* Class manipulation (`classList.add/remove`)
* Tab switching logic

---

## 💻 How It Works

1. Initially, only the **About** tab is visible
2. When a button is clicked:

   * All tabs are hidden using `display: none`
   * The selected tab is shown using `display: block`
   * The clicked button gets highlighted using a CSS class
