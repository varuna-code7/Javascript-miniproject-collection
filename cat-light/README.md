# 🐱 Cat Light Toggle (ON/OFF Project)

## 📌 Overview

This is a simple JavaScript mini project that simulates a light ON/OFF system with a cat illustration.
When the light is turned ON, the bulb glows and the cat becomes visible. When turned OFF, the scene goes dark only cat eye is visible.

---

## 🚀 Features

* 💡 Toggle light ON and OFF
* 🐱 Interactive cat visual change
* 🎨 Simple and clean UI
* ⚡ Real-time DOM manipulation using JavaScript

---

## 🛠️ Tech Stack

* CSS
* JavaScript (DOM Manipulation)

## 🎯 How It Works

* The ON and OFF buttons trigger JavaScript functions
* JavaScript updates:

  * Image source (bulb & cat)
  * Background color
  * Text status (Switched On / Off)
