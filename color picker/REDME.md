

# 🎨 Color Picker 

A simple and interactive **Color Picker ** built using HTML, CSS, and JavaScript. Users can change the background color of the page by clicking on different color buttons.

---

## 🚀 Features

* 🎯 Change background color with a single click
* 🎨 Multiple predefined color options
* 🖥️ Real-time color update display
* 📱 Responsive design using Bootstrap

---

* Bootstrap

## 🧠 How It Works

1. The page contains multiple buttons with different colors.
2. When a user clicks a button:

   * The background color of the container changes.
   * The selected color code is displayed on the screen.
3. Each button triggers a JavaScript function:

   * `one()` → sets color `#e0e0e0`
   * `two()` → sets color `#6fcf97`
   * `three()` → sets color `#56ccf2`
   * `four()` → sets color `#bb6bd9`
