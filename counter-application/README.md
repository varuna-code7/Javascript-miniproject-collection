# 🔢 Counter Application

## 📌 Overview

This is a simple and interactive Counter Application built using JavaScript.
It allows users to increase, decrease, and reset a numeric value dynamically.

---

## 🚀 Features

* ➕ Increase the counter value
* ➖ Decrease the counter value
* 🔄 Reset the counter to zero
* 🎨 Dynamic color change:

  * Green → Positive values
  * Red → Negative values
  * Black → Zero

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (DOM Manipulation)
* Bootstrap (for basic styling)

---

## 📂 Project Structure

```
counter-app/
│
├── index.html
├── style.css
└── manipulate.js
```

---

## 🎯 How It Works

* The counter value is displayed on the screen
* Button clicks trigger JavaScript functions:

  * `onIncrease()` → increments value
  * `onDecrease()` → decrements value
  * `onReset()` → resets value
* The text color updates based on the value

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the `counter-app` folder
3. Double-click `index.html` to run in browser
